# 3. Weather Dashboard (Streamlit + API)

Real-time weather data visualization with forecasts and interactive charts.

## Features
- Real-time weather data from OpenWeatherMap API
- Temperature, humidity, wind speed display
- 7-day forecast
- Interactive charts with Plotly
- Search by city name

## Tech Stack
- Streamlit (UI)
- requests (API calls)
- Plotly (visualization)
- OpenWeatherMap API (free tier)

## Setup
```bash
cd code
pip install -r requirements.txt
streamlit run app.py
```

Get your API key: [OpenWeatherMap](https://openweathermap.org/api)

## Deployment
Deploy to Streamlit Cloud
