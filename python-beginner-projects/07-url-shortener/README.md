# 7. URL Shortener (Flask + SQLite)

Create short, memorable links with QR code generation.

## Features
- Shorten long URLs
- Generate QR codes
- Click tracking
- Custom short codes
- Link expiration
- SQLite storage

## Tech Stack
- Flask (backend)
- SQLite (database)
- qrcode (QR generation)
- Python (URL encoding)

## Setup
```bash
cd code
pip install -r requirements.txt
python app.py
```

Visit: http://localhost:5000

## Deployment
Deploy to Render
