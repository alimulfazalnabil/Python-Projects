# 2. Smart To-Do App (Streamlit)

A simple yet powerful to-do app with task prioritization and category management.

## Features
- Add, edit, delete tasks
- Priority levels (High, Medium, Low)
- Category organization
- Task completion tracking
- Data persistence with CSV

## Tech Stack
- Streamlit (UI)
- Pandas (data handling)
- CSV (storage)

## Setup
```bash
cd code
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Deploy to Streamlit Cloud: [Streamlit Cloud](https://streamlit.io/cloud)
