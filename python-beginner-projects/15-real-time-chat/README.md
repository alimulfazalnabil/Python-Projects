# 15. Real-time Chat (Flask-SocketIO)

A real-time chat application with multiple users and rooms.

## Features
- Real-time messaging
- User nicknames
- Chat rooms
- User list
- Online status
- Message timestamps
- Connection/disconnection notifications

## Tech Stack
- Flask (backend)
- Flask-SocketIO (WebSocket support)
- JavaScript (frontend)

## Setup
```bash
cd code
pip install -r requirements.txt
python app.py
```

Visit: http://localhost:5000

## Deployment
Deploy to Render
