# 11. Pomodoro Timer (Streamlit)

A productivity timer using the Pomodoro technique with notifications.

## Features
- Customizable work/break intervals
- Session tracking
- Sound notifications
- Session history
- Progress visualization
- Task notes

## Tech Stack
- Streamlit (UI)
- time module (timing)

## Setup
```bash
cd code
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Deploy to Streamlit Cloud
