# 10. Snake Game (Pygame)

Classic snake game with score tracking and difficulty levels.

## Features
- Classic snake gameplay
- Score tracking
- Difficulty levels
- Game over detection
- Keyboard controls
- High score storage

## Tech Stack
- Pygame (game engine)

## Setup
```bash
cd code
pip install -r requirements.txt
python app.py
```

## Controls
- Arrow keys to move
- ESC to quit

## Deployment
Deploy web version to Pygbag
