# 5. Expense Tracker (Streamlit + SQLite)

Track your expenses with categories, visualizations, and budget management.

## Features
- Add/delete expenses
- Categorized spending
- Visual spending breakdown (pie charts)
- Monthly reports
- Budget alerts
- SQLite database persistence

## Tech Stack
- Streamlit (UI)
- SQLite (database)
- Pandas (data analysis)
- Plotly (charts)

## Setup
```bash
cd code
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Deploy to Streamlit Cloud
