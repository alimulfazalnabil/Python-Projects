# 13. Bulk File Organizer (Streamlit)

Organize files by type, date, or custom rules. Works locally or on Replit.

## Features
- Organize files by extension
- Organize by creation date
- Custom organization rules
- Undo operations
- File preview
- Dry run mode

## Tech Stack
- Streamlit (UI)
- os & shutil modules (file operations)
- pathlib (path handling)

## Setup
```bash
cd code
pip install -r requirements.txt
streamlit run app.py
```

## Usage
Run locally to organize files on your machine

## Deployment
Deploy to Replit or run locally
